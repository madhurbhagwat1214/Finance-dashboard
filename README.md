# Finance Performance Dashboard | Power BI

## Project Overview

This project presents an interactive Finance Performance Dashboard built in Power BI to analyze revenue, expenses, profitability, and financial trends. The dashboard helps stakeholders monitor financial performance, identify high-performing regions and expense categories, and track profit margins over time.

---

## Objectives

* Monitor overall financial performance
* Analyze revenue and expense trends
* Evaluate profitability across regions
* Identify high-cost expense categories
* Track profit margin performance over time
* Support data-driven financial decision-making

---

## Dataset Information

* Records: 5,000+
* Fields:

  * Transaction ID
  * Date
  * Region
  * Expense Category
  * Revenue
  * Expense
  * Profit
  * Year
  * Month
  * Month Number

## Key Insights

* North region generated the highest overall profit.
* IT category contributed significantly to expenses.
* Profit margin showed an upward trend over time.
* Financial performance remained stable across reporting periods.
* Regional profitability analysis highlighted areas for optimization.
